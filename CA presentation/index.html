Cache Coherence

In a multiprocessor system, data inconsistency may occur among adjacent levels or within the same level of the memory hierarchy. In a shared memory multiprocessor with a separate cache memory for each processor, it is possible to have many copies of any one instruction operand: one copy in the main memory and one in each cache memory. When one copy of an operand is changed, the other copies of the operand must be changed also.

Example :

Cache and the main memory may have inconsistent copies of the same object.

Suppose there are three processors, each having cache. Suppose the following scenario:-

  • Processor 1 read X : obtains 24 from the memory and caches it.
  • Processor 2 read X : obtains 24 from memory and caches it.
  • Again, processor 1 writes as X : 64, Its locally cached copy is updated. Now, processor 3 reads X, what value should it get?
  • Memory and processor 2 thinks it is 24 and processor 1 thinks it is 64.

There are two main types of cache coherence protocols:

1. Directory-based protocols:

This protocol uses a central directory to keep track of which processor has a copy of a memory location. When a processor modifies a location, it informs the directory and other processors are informed to invalidate their copy of that location.

2. Snooping protocols:

This protocol allows processors to snoop on the memory bus to monitor for modifications to shared locations. When a processor modifies a location, all other processors snoop the bus and invalidate their copy of that location.

Examples of cache coherence protocols include:

1. MSI (Modified, Shared, Invalid):

used for single-processor systems

2. MESI (Modified, Exclusive, Shared, Invalid):

used for multi-processor systems

3. MOESI (Modified, Owned, Exclusive, Shared, Invalid):

used for multi-processor systems

Conclusion

  • Cache coherence is an important aspect of computer architecture that ensures data consistency and improves system performance.

  • Cache coherence is a mechanism that ensures that all copies of a shared resource are consistent with each other.

  • Two main types of cache coherence protocols are directory-based and snooping protocols.

  • Examples of cache coherence protocols are MSI, MESI, MOESI.

  • Cache coherence is an important aspect of computer architecture that ensures data consistency and improves system performance.
