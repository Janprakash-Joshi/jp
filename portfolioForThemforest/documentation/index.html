Customization:

  1. Change Images:

​               At first replace all my images with your images.

              Do dot replace the 'portfolio' & 'portfolio-1' image if you need the same logo at the top.

  •  Home section 

​              Open the index.html file and go to line 114 from where home section is started , you can edit name      and intro text there and at line 128 change the image src with your image.

  • Social Media Section

             Go to line 135 you can see the social media section there just replace the href link of the a tag with your social media link.

  • About section

           Edit image url at line 167

           Edit about content at line 175

  • Services / Hire Us section

           Change the image of this section by css file, go to the style.css file under style folder and go to line 251 where you can replace your image name.

  • Portfolio Section

​        At line 223 and 230 at index.html you can edit image src with your image name.

  • Contact section 

     At line 269 and 275 you can edit email address.

  • Footer section

 from line 285 to 301 you can edit your social medial link as example link suggested in a tag.

And Finally change the copy right section at line 309 and  image source at the line 318.
